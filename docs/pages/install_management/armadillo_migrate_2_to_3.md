# Migrate Armadillo 2 to Armadillo 3

Migration support from Armadillo 2 to 3 has been removed as of Armadillo 5.13.2.
If you still need to perform this migration, the migration scripts and documentation
can be found in the
[Armadillo v3.4.0 release](https://github.com/molgenis/molgenis-service-armadillo/tree/v3.4.0/scripts/upgrade).
